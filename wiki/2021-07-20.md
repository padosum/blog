### Vue.js 
#### Filters  
- 데이터 포맷팅용으로 사용  
```javascript
export default {
  filters: {
	  formatDate(value) {
		  return new Date(value) 
		}
	}
	...
}
- `v-bind`나 중괄호 사용시  
```html
{{ message | formatDate }} 

<div v-bind:id="rawId" | formatDate"></div>
```
- **필터를 전역으로 사용하려는 경우** 
  1. 파일을 따로 분리해서 export  
	2. `main.js`에서 import 후 Vue 인스턴스 생성 전에 `Vue.filter` 로 사용  
- 적용 후 확인을 해보니 서버를 껐다가 켜야 적용이 된다.
