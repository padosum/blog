### [[React]]
props을 [[JavaScript-Spread-syntax|스프레드 연산자]]를 사용해서 JSX에 전달할 수 있다.
아래와 같이 `input`이라는 props은 [[JavaScript-Object|object]] 형식이기 때문에 
```javascript
<Input
        label="Amount"
        input={{
          id: 'amount_' + props.id,
          type: 'number',
          min: '1',
          max: '5',
          step: '1',
          defaultValue: '1',
        }}
      />
```
다음과 같이 간단하게 전달할 수 있다.
```javascript
const Input = (props) => {
  return (
	  <div>
	    <label htmlFor={props.input.id}>{props.label}</label>
	    <input {...props.input} />
	  </div>
	)
}
```
