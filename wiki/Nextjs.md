- [[React]]를 기반으로 하는 [[framework]]
- 코드를 작성하는 방법, 프로젝트 구성 방법에 규칙/지침이 있다.
- React 앱을 사용하면서 추가하게 되는 기능들을 기본적으로 추가해뒀다.
- [[SSR]] 기능을 내장하고 있다.
- 파일 기반으로 라우팅을 한다. (Pages Router)
	- App Router도 있다.
- Node.js 코드를 이용해 React 프로젝트에 백엔드 API를 추가할 수 있다.
	- 따로 백엔드 API 프로젝트를 구축하지 않아도 된다.
