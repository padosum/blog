### JavaScript 배열, 객체 복사하기
배열 복사 (얕은 복사)
```javascript
const arr = [1, 2, 3]
const copyArr = arr.slice()
```

객체 복사를 위해 `lodash`의 `_.cloneDeep()`을 사용하는 것을 많이 봤다.
```javascript
import cloneDeep from 'lodash/cloneDeep.js'
const original = {
  name: 'padosum',
  language: 'korean'
}
const result = cloneDeep(original)
```
