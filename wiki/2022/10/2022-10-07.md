## 📚 오늘 도전하고, 배운 것

### mac os에서 `rar` 파일 압축 해제하는 방법
homebrew를 사용해 `rar` 설치
```bash
$ brew install rar
```

다음과 같이 매우 간편하게 사용할 수 있었다. [출처](https://superuser.com/questions/52124/how-can-i-extract-rar-files-on-the-mac)
```bash
$ unrar x <filename>
```
