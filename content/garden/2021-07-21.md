## Note
- Codeup 100제 6096  
```python
d = [] 
for i in range(19):
  a = list(map(int, input().split()))
  d.append(a)

t = int(input())  
for i in range(t):
  x, y = map(int, input().split())
  for j in range(19): 
    for k in range(19):
      if k == y-1:
        d[j][k] = int(not bool(d[j][k]))
      if j == x-1:
        d[j][k] = int(not bool(d[j][k]))
for i in range(19):
  p = ''
  for j in range(19):
    p = p + str(d[i][j]) + ' '
  print(p)  
```
- x, y가 왜 매번 헷갈리는지 모르겠다. 주의해야 한다.
