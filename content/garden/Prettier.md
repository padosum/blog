**Prettier, 프리티어**
- 코드를 일관적인 스타일로 다듬어준다.  
  
## Prettier 설치  
```bash
npm i -D prettier 
```
## Prettier 사용 
```bash
npx prettier app.js --write
```
- `--write` 옵션은 파일을 재작성, 옵션이 없으면 결과를 터미널에 출력
