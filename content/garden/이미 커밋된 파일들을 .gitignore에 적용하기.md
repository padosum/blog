[[Git]]을 사용하면서 `.gitignore`을 이용해 커밋하고 싶지 않은 파일들을 설정할 수 있는데 어제의 나처럼 이미 커밋을 해버려서 `.gitignore`에 추가해도 무시되지 않을 수 있다. 그럴 때 사용하는 방법  

우선 `.gitignore`에 추가하고 싶은 파일을 넣고 수정한다.  
```bash 
$ git rm -r --cached . 
$ git add . 
``` 
그 다음 평소처럼 커밋하면 된다.
