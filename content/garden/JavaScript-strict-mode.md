## strict mode ?? 
- ES5 부터 지원된 엄격 모드(strict mode)
	- 자바스크립트 언어 문법을 좀 더 엄격히 적용해서 오류를 발생시킬 가능성을 높이거나 엔진 최적화 작업에 문제를 일으킬 수 있는 코드에 대해 명시적인 에러를 발생시킨다.  
- ESLint(린트 도구)를 사용하면 strict mode와 유사한 효과를 얻을 수 있다.
