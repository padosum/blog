### Python  
```python
a = input() # a라는 변수에 값을 받기  
print(a) # 변수a의 값 출력하기

a, b = input().split() # 공백으로 나눠서 입력한 값 각각 변수에 담기  
print(a, b) # 사이에 공백을 넣고 각 값을 출력  

a = ord(input()) # 문자의 유니코드 값을 돌려주는 함수 

a, b = input().split()
print(a, b, sep=":") # a:b 로 출력됨  
```
